# Textin xParse — 面向 Agent 与 RAG 的新一代文档处理 AI Infra

欢迎来到 Textin xParse！我们获得**全球 1,000+ 领先企业的信任和选择**，已处理**超 1,000,000,000+ 页文档**。

Textin xParse 将复杂文档（PDF / Office / 图片等）转成**可追溯、可引用、可用于 Agent 与 RAG 的结构化数据**，让文档处理像流水线一样简单。通过统一的标准协议与丰富的 AI 生态集成，为 RAG、Agent 等 AI 应用提供强大的数据底座。**精准，高效，稳定。**

要开始使用，请查看我们的开源项目：

- [`xparse-python-sdk`](https://github.com/intsig-textin/xparse-python-sdk) - xParse API 的 Python SDK，推荐作为你在 Python 项目里调用 xParse 的默认入口。
- [`xparse-kit/visualizer`](https://github.com/intsig-textin/xparse-kit/tree/master/packages/visualizer) - 结果回溯可视化组件：支持多页图片渲染、缩放、旋转、标记框绘制、高亮与点击交互，适用于文档解析校对、RAG 分 chunk 回溯等场景。
- [`xparse-dify-plugin`](https://github.com/intsig-textin/xparse-dify-plugin) - Dify 工具插件：将 PDF/Word/Excel/PPT/图片等文件解析成可查询、可分析的结构化元素与元数据，便于在 Dify 工作流里直接接入。
- [`langchain-xparse`](https://github.com/intsig-textin/langchain-xparse) - xParse 的 LangChain 集成，为大模型和 Agent 应用强大的文档解析能力。
